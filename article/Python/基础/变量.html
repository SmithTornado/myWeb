Python变量

变量是Python中存储数据的容器，无需提前声明类型，直接赋值即可使用，值和类型可动态修改，是程序处理数据的核心载体。

一、变量定义与赋值

语法：变量名 = 值，=为赋值符，将右侧值存入左侧变量容器。

# 1. 基础赋值：不同类型变量定义
num = 100          # 整数类型
price = 99.9       # 浮点数类型
name = "小明"      # 字符串类型
is_valid = True    # 布尔类型（True/False）

# 2. 打印变量值
print(num)         # 输出：100
print(name)        # 输出：小明

# 3. 一行定义多个变量
a, b, c = 1, 2, 3  # 多变量分别赋值
x = y = 5          # 多变量赋相同值
print(a, b, c)     # 输出：1 2 3
print(x, y)        # 输出：5 5

二、变量命名规则

1. 必遵守规则

  • 由字母、数字、下划线组成，不可含空格/特殊符号（@、#等）
  • 不能以数字开头，不能使用Python关键字（if、for、print等）
  • 区分大小写（name和Name是不同变量）

2. 推荐规范

  • 蛇形命名法：小写字母+下划线（如user_id、total_price）
  • 见名知义：变量名体现数据含义（age存年龄、phone存手机号）
# 合法变量名（推荐）
user_name = "小红"
age_18 = 18
score = 95

# 非法变量名（报错）
# 123num = 20  # 以数字开头
# user@name = "小刚"  # 含特殊符号
# if = 5  # 使用关键字

三、变量常见数据类型

用type(变量名)可查看变量类型，常用基础类型如下：

 1. 整数（int）：正负数、0
num1 = 20
num2 = -5
print(type(num1))  # 输出：<class 'int'>

# 2. 浮点数（float）：带小数的数
f1 = 19.9
f2 = 30.0
print(type(f1))    # 输出：<class 'float'>

# 3. 字符串（str）：文本，单/双引号包裹
s1 = 'Python'
s2 = "编程学习"
print(type(s1))    # 输出：<class 'str'>

# 4. 布尔（bool）：仅True（真）、False（假）
b1 = True
b2 = False
print(type(b1))    # 输出：<class 'bool'>

四、变量核心操作

1. 重新赋值

变量值和类型可随时修改，新值会覆盖旧值。

2. 变量运算

python# 数值运算（加减乘除）
a = 10
b = 3
print(a + b)  # 加法：13
print(a * b)  # 乘法：30
print(a / b)  # 除法：3.3333333333333335

# 字符串拼接（+号）
s1 = "Hello"
s2 = "Python"
print(s1 + " " + s2)  # 输出：Hello Python

五、注意事项

  • 变量必须先赋值再使用，未赋值直接调用会报错
  • =是赋值，判断是否相等用==（如a == b）
  • 不与Python内置函数重名（如不定义print = 10，会覆盖打印功能）
